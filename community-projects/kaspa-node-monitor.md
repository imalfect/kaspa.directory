# Kaspa Node Monitor

## Project Information
- Creator: iMalFect
- Category: Node Utilities
- Created in 2023
- Programming Language: JavaScript
- License: ISC
## Links
- [GitHub](https://github.com/imalfect/KaspaNodeMonitor)

## Project Description
Kaspa Node Monitor is a tool that lets you monitor your Kaspa node.
The node monitor features an easy-to-use CLI tool that lets you manage your installation

Information displayed:
- Block count
- Header count
- Peers count
- DAA Score
- Blue score
- Node version
- Node sync status
- Node network
- Network difficulty
- Hardware information (this can be disabled), such as:
    - Hostname
    - Location
    - CPU Model
    - CPU Thread Count
    - RAM (Free & Total)
    - System load
